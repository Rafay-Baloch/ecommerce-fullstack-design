# 🛒 Ecommerce Full-Stack Application
This project is part of my Full-Stack Web Development Internship. It's a professional, database-driven platform built using the MERN stack.

## 📅 Project Roadmap & Progress

### 🟢 Week 1: Project Setup & Static Frontend (Completed)
**Goal:** Build the foundation and responsive UI matching the Figma design.
- [x] Development environment setup (Node.js, React, Express).
- [x] **Home Page:** Created featured sections and responsive layouts.
- [x] **Product Listing Page:** Implemented Grid and List view toggles.
- [x] **Product Details Page:** Designed a professional product showcase UI.
- [x] **Cart Page:** Created a clear summary and items list layout.
- [x] **Responsiveness:** Fully optimized for Mobile, Tablet, and Desktop using TailwindCSS.

### 🔵 Week 2: Backend Setup & Dynamic Integration (Completed)
**Goal:** Connect the frontend to a real database and implement core logic.
- [x] **Database:** Configured MongoDB to store and manage product data.
- [x] **REST API:** Developed Express.js endpoints for fetching products (CRUD).
- [x] **Dynamic Rendering:** Connected Frontend to Backend to show real data.
- [x] **Search Engine:** Implemented a real-time search bar to filter items.
- [x] **Sticky Sidebar:** Optimized UI for easy navigation through categories.
- [x] **Cart Logic:** Developed full cart management (Add, Remove, Clear All, and Counter).

### 🟡 Week 3: Auth, Admin CRUD & Persistence (Final Phase)
In this final phase, the core focus was on security, administration, and data persistence:

- [x] User Authentication: Implemented secure Login and Signup functionality using JWT (JSON Web Tokens) and password hashing.

- [x] Professional Admin Dashboard: Created a high-end dashboard for inventory management.

- [x] Full CRUD Operations: Admins can Create, Read, Update, and Delete products directly from the UI.

- [x] Protected Routes: Restricted access to sensitive admin pages to authorized users only.

- [x] Cart Persistence: Integrated LocalStorage with React Context API to ensure cart items remain saved even after a page refresh.

- [x]Responsive Design: Fully optimized for mobile, tablet, and desktop views using Tailwind CSS.

- [x] Version Control: Final source code committed and pushed to GitHub.
